# WFHAttendance

This repository contains the WFH Attendance System project.

## Structure

- `wfh-attendance-backend` → Backend service (API, database, authentication)

## Features

- Employee attendance tracking
- Work from home attendance management
- Backend REST API
- Database integration

## Tech Stack

Backend:
- Node.js
- Express.js
- MySQL

## Setup Backend

```bash
cd wfh-attendance-backend
npm install
npm run dev
